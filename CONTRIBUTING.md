# Contributing to Programmdle

## Reporting dataset errors

The fastest way to fix an inaccuracy is to open a **Dataset correction** issue using the template provided. Include a source link — corrections without references won't be merged.

## Proposing new entries

New entries must meet two criteria:
1. The technology is recognisable by name to a working developer without Googling it
2. All 8 attributes can be assigned unambiguously

Open an issue with the proposed entry's full attribute set before submitting a PR.

## Code contributions

1. Fork the repo
2. Create a branch: `feature/your-feature-name`
3. Open a PR against `main` with a clear description
4. CI must pass before review
